# Python 容器（Container）汇总

## Python 容器概览

Python 常用容器主要有：

| 容器    | 关键字   | 是否有序          | 是否可修改 | 是否允许重复   | 查询效率 |
| ----- | ----- | ------------- | ----- | -------- | ---- |
| List  | []    | 是             | 是     | 是        | O(n) |
| Tuple | ()    | 是             | 否     | 是        | O(n) |
| Dict  | {}    | 是(Python3.7+) | 是     | Key不允许重复 | O(1) |
| Set   | set() | 否             | 是     | 否        | O(1) |

---

# 1. List（列表）

## 定义

```python
nums = [1, 2, 3]
```

## 特点

| 项目   | 描述   |
| ---- | ---- |
| 底层结构 | 动态数组 |
| 有序   | 是    |
| 可修改  | 是    |
| 可重复  | 是    |
| 支持索引 | 是    |

## 优势

| 优势     | 说明     |
| ------ | ------ |
| 使用简单   | 最常用容器  |
| 支持索引   | O(1)访问 |
| 支持切片   | 数据处理方便 |
| 适合顺序存储 | 类似C数组  |

## 注意点

| 注意点     | 说明   |
| ------- | ---- |
| 中间插入慢   | O(n) |
| 删除元素慢   | O(n) |
| 大量查找效率低 | O(n) |

## 常见操作

```python
nums.append(4)
nums.remove(2)
nums.sort()
```

---

# 2. Tuple（元组）

## 定义

```python
point = (10, 20)
```

## 特点

| 项目   | 描述 |
| ---- | -- |
| 有序   | 是  |
| 可修改  | 否  |
| 可重复  | 是  |
| 支持索引 | 是  |

## 优势

| 优势          | 说明        |
| ----------- | --------- |
| 内存占用更小      | 比List节省内存 |
| 访问速度快       | 略快于List   |
| 数据安全        | 不会被误修改    |
| 可作为Dict Key | 因为不可变     |

## 注意点

| 注意点     | 说明               |
| ------- | ---------------- |
| 创建后不能修改 | append/remove不可用 |
| 单元素需加逗号 | (1,)             |

## 常见场景

```python
shape = (3, 224, 224)

name, age = ("Tom", 18)
```

AI项目中大量用于：

```python
(batch, channel, height, width)
```

# 3. Dict（字典）

## 定义

```python
student = {
    "name": "Tom",
    "score": 95
}
```

## 特点

| 项目          | 描述   |
| ----------- | ---- |
| Key-Value结构 | 是    |
| 有序          | 是    |
| Key唯一       | 是    |
| 查询速度        | O(1) |

## 优势

| 优势       | 说明     |
| -------- | ------ |
| 查询极快     | 哈希表实现  |
| 表达能力强    | 类似JSON |
| AI项目大量使用 | 配置管理   |

## 注意点

| 注意点      | 说明         |
| -------- | ---------- |
| Key必须可哈希 | List不能做Key |
| Key重复会覆盖 | 后值覆盖前值     |

## 常见操作

```python
student["age"] = 18

student.get("score")

student.keys()

student.values()
```

## 对应JSON

```json
{
    "name":"Tom",
    "score":95
}
```

# 4. Set（集合）

## 定义

```python
nums = {1, 2, 3}
```

## 特点

| 项目   | 描述   |
| ---- | ---- |
| 无序   | 是    |
| 不重复  | 是    |
| 可修改  | 是    |
| 查询速度 | O(1) |

## 优势

| 优势     | 说明       |
| ------ | -------- |
| 自动去重   | 非常方便     |
| 查找速度快  | 哈希实现     |
| 集合运算丰富 | 交集、并集、差集 |

## 注意点

| 注意点     | 说明          |
| ------- | ----------- |
| 无索引     | nums[0]错误   |
| 无序      | 不保证顺序       |
| 元素必须可哈希 | List不能放入Set |

## 常见操作

```python
nums.add(4)

nums.remove(2)
```

## 去重神器

```python
nums = [1,1,2,2,3]

result = list(set(nums))
```

结果：

```python
[1,2,3]
```

# 5. 容器效率对比

| 操作   | List | Tuple | Dict | Set  |
| ---- | ---- | ----- | ---- | ---- |
| 索引访问 | O(1) | O(1)  | -    | -    |
| 查找元素 | O(n) | O(n)  | O(1) | O(1) |
| 插入尾部 | O(1) | 不支持   | O(1) | O(1) |
| 删除元素 | O(n) | 不支持   | O(1) | O(1) |
