# Python 数据类型总表（仅数据类型）

> Python 3.x（CPython参考实现）

---

## 数据类型汇总

| 类型 | 分类 | 是否可变 | 内存/大小特点 | 取值范围 | 说明 |
|------|------|----------|----------------|----------|------|
| int | 数值 | ❌ | 动态大小（约28B起） | 无限精度（仅受内存限制） | 整数类型，小整数缓存(-5~256) |
| float | 数值 | ❌ | 8 bytes（IEEE 754） | ±1.7e308 | 双精度浮点 |
| complex | 数值 | ❌ | 16 bytes | float范围 | 复数（实部+虚部） |
| bool | 布尔 | ❌ | ~28 bytes | True / False | int子类 |
| str | 文本 | ❌ | 动态（≈49B+） | Unicode | 不可变字符串 |
| bytes | 二进制 | ❌ | 33B+ | 0~255 | 不可变字节序列 |
| bytearray | 二进制 | ✔️ | 动态 | 0~255 | 可变字节序列 |
| memoryview | 二进制 | ✔️(视图) | 轻量引用 | 0~255 | 零拷贝访问内存 |
