# 激活函数（Activation Function）

---

## 2. 为什么需要激活函数

假设没有激活函数：

```text
输入层
   ↓
线性层
   ↓
线性层
   ↓
线性层
   ↓
输出层
```

数学上：

```math
y=W_3(W_2(W_1x))
```

展开后仍然是：

```math
y=Wx
```

无论堆叠多少层，本质上还是一个线性模型。

---

因此：

```text
没有激活函数
=
深层网络退化成单层线性模型
```

无法学习复杂规律。

---

## 3. 激活函数的作用

### 引入非线性

现实世界中的大多数问题都是非线性的：

- 图像识别
- 语音识别
- 自然语言处理
- 自动驾驶

激活函数让神经网络能够拟合复杂的非线性关系。

---

### 提升模型表达能力

例如：

```text
感知机
    ↓
只能解决线性可分问题
    ↓
无法解决 XOR
```

加入激活函数和隐藏层后：

```text
多层感知机(MLP)
    ↓
可以解决 XOR
```

## 9. 常见激活函数比较

| 激活函数 | 输出范围 | 是否非线性 | 梯度消失 | 使用场景 |
|----------|----------|----------|----------|----------|
| Sigmoid | (0,1) | 是 | 严重 | 二分类输出 |
| Tanh | (-1,1) | 是 | 存在 | 早期网络 |
| ReLU | [0,+∞) | 是 | 较少 | 隐藏层首选 |
| Leaky ReLU | (-∞,+∞) | 是 | 更少 | 深层网络 |
| Softmax | 概率分布 | 是 | 无 | 多分类输出 |

## 10. 深度学习中的使用原则

### 二分类

```python
nn.Sigmoid()
```

输出：

```text
属于正类的概率
```

---

### 多分类

```python
nn.Softmax()
```

或者：

```python
nn.CrossEntropyLoss()
```

（内部已包含 Softmax）

---

### 隐藏层

绝大多数情况下：

```python
nn.ReLU()
```

例如：

```python
net = nn.Sequential(
    nn.Linear(784, 256),
    nn.ReLU(),
    nn.Linear(256, 10)
)
```

## 总结

激活函数是神经网络学习非线性规律的关键。

```text
输入
 ↓
wx+b
 ↓
激活函数
 ↓
输出
```

常用选择：

```text
隐藏层：
    ReLU

二分类输出层：
    Sigmoid

多分类输出层：
    Softmax
```

没有激活函数：

```text
神经网络 = 线性模型
```

有了激活函数：

```text
神经网络 = 强大的非线性拟合器
```

这也是深度学习能够识别图像、语音和自然语言的核心原因之一。
