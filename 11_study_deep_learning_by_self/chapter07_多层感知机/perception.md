# 感知机（Perceptron）详解

## 1. 什么是感知机

感知机（Perceptron）是机器学习历史上最早的人工神经网络模型之一，由 Frank Rosenblatt 于 1957 年提出。

感知机本质上是一个：

```text
线性二分类模型
```

它的作用是：

```text
输入一组特征
    ↓
计算加权和
    ↓
判断属于哪一类
```

例如：

- 判断邮件是否为垃圾邮件
- 判断图片中是否有猫
- 判断贷款申请是否通过
- 判断数字是否大于5

感知机也是现代神经网络和深度学习的起点。

# 2. 生物神经元与感知机

## 生物神经元

```text
树突
 ↓
接收信号
 ↓
细胞体
 ↓
判断是否激活
 ↓
轴突
 ↓
输出信号
```

---

## 感知机

```text
x1
 \
  \
x2 ---> Σ ---> 激活函数 ---> y
  /
 /
x3
```

其中：

```text
x1,x2,x3
```

是输入特征。

```text
w1,w2,w3
```

是权重。

```text
b
```

是偏置（Bias）。

感知机模仿了生物神经元的工作过程。

---

# 3. 感知机数学模型

感知机首先计算：



即：

```math
z=w_1x_1+w_2x_2+\cdots+w_nx_n+b
```

---

然后经过激活函数：

```math
y = sign(z)
```

即：

```math
y=
\begin{cases}
1,& z>0 \\
-1,& z\le0
\end{cases}
```

或者：

```math
y=
\begin{cases}
1,& z>0 \\
0,& z\le0
\end{cases}
```

感知机根据加权和是否超过阈值来决定输出类别。

# 4. 一个简单例子

## 判断是否通过考试

特征：

```text
x1 = 数学成绩
x2 = 英语成绩
```

假设：

```text
w1 = 0.6
w2 = 0.4
b = -60
```

---

学生A：

```text
数学 = 80
英语 = 90
```

计算：

```math
z=0.6\times80+0.4\times90-60
```

```math
=48+36-60
```

```math
=24
```

由于：

```math
24>0
```

输出：

```text
1
```

表示：

```text
通过
```

---

学生B：

```text
数学 = 40
英语 = 50
```

计算：

```math
z=0.6\times40+0.4\times50-60
```

```math
=24+20-60
```

```math
=-16
```

输出：

```text
0
```

表示：

```text
不通过
```

# 5. 几何意义

感知机实际上是在寻找一条分界线。

二维情况：

```math
w_1x_1+w_2x_2+b=0
```

对应：

```text
一条直线
```

---

例如：

```math
2x_1+x_2-5=0
```

图示：

```text
      ○ ○ ○
     ○ ○ ○

----------------

 × × × × ×
 × × × × ×
```

感知机的任务：

```text
找到一条线
把两类数据分开
```

---

三维空间：

```math
w^Tx+b=0
```

对应：

```text
一个平面
```

---

更高维空间：

```text
超平面(Hyperplane)
```

感知机的本质就是寻找最优分类超平面。

# 6. 感知机学习过程

假设：

```text
真实标签：
y

预测标签：
ŷ
```

---

训练流程：

```text
输入样本
    ↓
预测结果
    ↓
比较误差
    ↓
更新权重
    ↓
继续训练
```

---

## 更新规则

如果分类错误：

```math
w=w+\eta yx
```

```math
b=b+\eta y
```

其中：

```text
η
```

称为：

```text
学习率（Learning Rate）
```

---

# 7. 感知机训练算法

初始化：

```text
w = 0
b = 0
```

---

循环：

```text
取一个训练样本
    ↓
预测
    ↓
是否分类正确
```

---

正确：

```text
不更新
```

---

错误：

```text
更新参数
```

---

直到：

```text
所有样本分类正确
```

或者：

```text
达到最大迭代次数
```
