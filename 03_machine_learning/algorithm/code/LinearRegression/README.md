# 线性回归（Linear Regression）入门 Demo

## 1. 项目简介

本项目通过一个简单的房价预测案例演示机器学习中最基础的回归问题。

目标：

根据房屋面积预测房价。

---

## 2. 项目原理

假设房价与面积之间存在线性关系：

```text
面积越大
房价越高
```

使用线性回归模型：

```text
y = wx + b
```

其中：

| 参数 | 含义   |
| -- | ---- |
| x  | 房屋面积 |
| y  | 房屋价格 |
| w  | 权重   |
| b  | 偏置   |

机器学习训练的目标：

```text
找到最优的 w 和 b
```

使预测结果尽可能接近真实值。

---

## 3. 训练数据

| 面积(㎡) | 房价(万) |
| ----- | ----- |
| 50    | 100   |
| 60    | 120   |
| 70    | 140   |
| 80    | 160   |
| 90    | 180   |

观察数据可以发现：

```text
面积每增加10㎡

房价增加20万
```

因此理论关系为：

```text
房价 = 面积 × 2
```

---

## 4. 代码实现

```python
import numpy as np
import matplotlib.pyplot as plt
from sklearn.linear_model import LinearRegression

# -------------------------
# 训练数据
# -------------------------

X = np.array([
    [50],
    [60],
    [70],
    [80],
    [90]
])

y = np.array([
    100,
    120,
    140,
    160,
    180
])

# -------------------------
# 创建模型
# -------------------------

model = LinearRegression()

# -------------------------
# 模型训练
# -------------------------

model.fit(X, y)

# -------------------------
# 输出模型参数
# -------------------------

print("权重 =", model.coef_[0])
print("偏置 =", model.intercept_)

# -------------------------
# 预测
# -------------------------

area = np.array([[75]])

price = model.predict(area)

print("75㎡房价 =", price[0])

# -------------------------
# 可视化
# -------------------------

plt.scatter(X, y, label="Train Data")

plt.plot(
    X,
    model.predict(X),
    label="Regression Line"
)

plt.xlabel("Area")
plt.ylabel("Price")

plt.legend()

plt.show()
```
