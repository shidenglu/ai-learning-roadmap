# 1. 什么是逻辑回归

逻辑回归（Logistic Regression）是一种经典的机器学习分类算法。

虽然名字叫：

```text
逻辑回归（Regression）
```

但实际上用于：

```text
分类（Classification）
```

例如：

* 垃圾邮件识别
* 猫狗分类
* 用户是否购买商品
* 用户是否点击广告
* 疾病诊断

---

# 2. 为什么需要逻辑回归

假设：

```text
输出：
是否及格
```

数据：

| 成绩 | 是否及格 |
| -- | ---- |
| 40 | 0    |

其中：

```text
0 = 不及格
1 = 及格
```

---

我们的目标：

```text
给定成绩

预测：

属于哪一类
```

# 3. 线性回归的问题

线性回归模型：

```text
y = wx + b
```

例如：

```text
y = 0.05x - 2
```

输入：

```text
x = 100
```

输出：

```text
y = 3
```

---

问题：

分类结果应该是：

```text
0 或 1
```

但是线性回归可能输出：

```text
-3
2.5
10
100
```

完全不符合概率要求。

---

因此需要一个函数：

```text
输入：
任意实数

输出：
0~1之间
```

这就是 Sigmoid 函数。

---

# 4. Sigmoid函数

公式：

[
\sigma(z)=\frac{1}{1+e^{-z}}
]

特点：

```text
输入：
(-∞,+∞)

输出：
(0,1)
```

---

当：

```text
z = -10
```

输出：

```text
≈0
```

---

当：

```text
z = 0
```

输出：

```text
0.5
```

---

当：

```text
z = 10
```

输出：

```text
≈1
```

---

图像：

```text
         1
         |
      ___|
    /
  /
 /
|
+----------------
|
|
|
         0
```

S形曲线。

---

# 5. 逻辑回归模型

首先：

```text
z = wx+b
```

与线性回归一样。

---

然后：

```text
p = sigmoid(z)
```

即：

[
p=\frac1{1+e^{-z}}
]

---

完整模型：

[
p(y=1|x)
========

\frac1{1+e^{-(wx+b)}}
]

含义：

```text
属于类别1的概率
```

---

# 6. 概率输出

假设：

```text
p = 0.9
```

表示：

```text
90%概率属于类别1
```

---

假设：

```text
p = 0.1
```

表示：

```text
10%概率属于类别1
```

---

分类规则：

```text
p > 0.5
    → 1

p <= 0.5
    → 0
```

---

# 7. 损失函数

不能使用 MSE。

因为：

```text
分类问题
+
Sigmoid函数
```

使用 MSE 训练效果较差。

---

逻辑回归采用：

```text
交叉熵损失
```

Cross Entropy

---

公式：

[
L
=

## -y\log(p)

(1-y)\log(1-p)
]

---

当：

```text
真实值=1

预测概率=0.99
```

损失：

```text
很小
```

---

当：

```text
真实值=1

预测概率=0.01
```

损失：

```text
非常大
```

---

符合训练需求。

---

# 8. 梯度下降

目标：

```text
寻找最佳参数

w
b
```

---

步骤：

## Step1

随机初始化

```text
w
b
```

---

## Step2

计算预测

```text
z = wx+b
```

```text
p = sigmoid(z)
```

---

## Step3

计算损失

```text
CrossEntropy
```

---

## Step4

计算梯度

```text
dw
db
```

---

## Step5

更新参数

[
w=w-\eta dw
]

[
b=b-\eta db
]

---

重复：

```text
Step2~Step5
```

直到损失最小。

# 9. NumPy实现逻辑回归

## 导入库

```python
import numpy as np
```

---

## 数据

```python
X = np.array([
    [40],
    [50],
    [60],
    [70],
    [80]
])

y = np.array([
    [0],
    [0],
    [1],
    [1],
    [1]
])
```

---

## Sigmoid

```python
def sigmoid(z):
    return 1/(1+np.exp(-z))
```

---

## 参数初始化

```python
w = 0.01
b = 0
```

---

## 前向传播

```python
z = X*w+b

p = sigmoid(z)
```

---

预测概率：

```python
print(p)
```

# 10. Scikit-Learn实现

安装：

```bash
pip install scikit-learn
```

---

代码：

```python
from sklearn.linear_model import LogisticRegression

X = [
    [40],
    [50],
    [60],
    [70],
    [80]
]

y = [
    0,
    0,
    1,
    1,
    1
]

model = LogisticRegression()

model.fit(X,y)
```

---

预测：

```python
model.predict([[65]])
```

输出：

```python
[1]
```

---

预测概率：

```python
model.predict_proba([[65]])
```

输出：

```python
[[0.12,0.88]]
```

表示：

```text
12%属于0

88%属于1
```

# 11. 逻辑回归与线性回归对比

| 项目   | 线性回归    | 逻辑回归          |
| ---- | ------- | ------------- |
| 任务   | 回归      | 分类            |
| 输出   | 任意实数    | 概率            |
| 模型   | wx+b    | sigmoid(wx+b) |
| 损失函数 | MSE     | CrossEntropy  |
| 输出范围 | (-∞,+∞) | (0,1)         |
| 典型应用 | 房价预测    | 垃圾邮件识别        |

# 13. 逻辑回归缺点

缺点：

```text
只能学习线性边界

复杂任务效果一般

表达能力有限
```

---

例如：

```text
猫狗图片分类
```

通常需要：

```text
CNN
```

而不是逻辑回归。

# 总结

逻辑回归本质上是：

```text
线性模型
+
Sigmoid函数
+
交叉熵损失
```

即：

线性回归：

y = wx + b

↓

逻辑回归：

z = wx + b

↓

Sigmoid

↓

输出概率

↓

完成分类任务

它是机器学习中第一个真正意义上的分类模型，也是理解神经网络输出层和交叉熵损失函数的基础。
