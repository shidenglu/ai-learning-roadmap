# 决策树核心原理：分叉提问是如何设置的？

决策树在构建时，设置分叉提问并不是随机的，而是遵循一个黄金法则：**“哪一个特征和切分点能让分出来的子集最‘纯’（即混乱度降得最低），就选择谁。”**

算法会对所有特征的所有可能取值进行“地毯式搜索”，并通过数学指标进行打分选拔。

---

## 1. 核心量化指标：不纯度（Impurity）

在算法眼中，“纯”意味着**一个盒子里的样本尽量属于同一种类**。为了量化这种混乱程度，最常用的指标是**基尼系数（Gini Impurity）**。

### 基尼系数公式
$$	ext{Gini} = 1 - \sum_{k=1}^K p_k^2$$
*(其中 $p_k$ 代表第 $k$ 个类别在当前节点中所占的比例)*

* **绝对纯（Gini = 0）**：盒子里 100% 都是好瓜。$	ext{Gini} = 1 - (1.0)^2 = 0$
* **极度混乱（Gini = 0.5）**：好瓜坏瓜各占 50%。$	ext{Gini} = 1 - (0.5^2 + 0.5^2) = 0.5$

---

## 2. 分类特征的选拔过程（以挑西瓜为例）

### 方案 A：根据“纹理”分叉（清晰 / 模糊）
* **左分支（清晰）**：60 个瓜（50 好，10 坏） $

* **左分支**：50 个瓜（25 好，25 坏） $

---

## 3. 连续数值特征的处理机制

如果特征是连续的数字（如“西瓜重量”：$3.2	ext{kg}, 5.1	ext{kg}, 2.7	ext{kg}\dots$），算法会通过以下三步“化连续为离散”：

1. **升序排列**：将所有样本的该特征值从小到大进行排序。
2. **寻找候选切分点**：取每两个相邻数字的**中间点**作为候选的提问切分点（例如：$2.5	ext{kg}$ 和 $3.0	ext{kg}$ 之间取 $2.75	ext{kg}$）。
3. **遍历打分**：把这些切分点挨个变成二分问题（如“重量 $\le 2.75	ext{kg}$ 吗？”），并计算它们分叉后的基尼系数，最终挑出能让不纯度降得最低的那个数字作为分叉点。

---

## 💡 核心总结

决策树设置分叉的过程，本质上是一场**基于贪心策略的“全特征选拔赛”**。
每一次分裂，算法都会穷举所有特征及所有可能的切分点，通过数学公式算出分裂后的不纯度。**谁能让数据瞬间变得最纯净、最有序，谁就能当选为该节点的分叉提问。**
