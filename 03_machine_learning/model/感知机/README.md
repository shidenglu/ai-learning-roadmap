# 感知机（Perceptron）介绍

## 1. 什么是感知机

感知机（Perceptron）是最早的人工神经网络模型之一，由 Frank Rosenblatt 于 1957 年提出。

它本质上是一个：

> 二分类线性模型

主要用于解决：

```text
是 / 否
正类 / 负类
0 / 1
```

等二分类问题。

感知机也是后续神经网络和深度学习的基础。

---

## 2. 感知机结构

一个感知机包含：

- 输入层(Input)
- 权重(Weight)
- 偏置(Bias)
- 激活函数(Activation Function)
- 输出(Output)

示意图：

```text
x1 ──►(w1)──┐
            │
x2 ──►(w2)──┤
            │
x3 ──►(w3)──┤──► Σ(wx+b) ─► Activation ─► y
            │
xn ──►(wn)──┘
```

---

## 3. 数学模型

### 第一步：线性组合

输入：

```math
x = [x_1,x_2,...,x_n]
```

权重：

```math
w = [w_1,w_2,...,w_n]
```

计算：

```math
z=w^Tx+b
```

展开：

```math
z=w_1x_1+w_2x_2+\cdots+w_nx_n+b
```

其中：

- w：权重
- b：偏置
- z：神经元输出

---

## 第二步：激活函数

感知机使用阶跃函数（Step Function）：

```math
f(z)=
\begin{cases}
1, & z\ge0 \\
0, & z<0
\end{cases}
```

输出：

```math
y=f(z)
```

---

## 4. 决策边界

对于二维输入：

```math
z=w_1x_1+w_2x_2+b
```

分类边界满足：

```math
w_1x_1+w_2x_2+b=0
```

这是一条直线。

因此感知机本质上是在寻找：

```text
一个超平面
```

将两类数据分开。

二维：

```text
直线
```

三维：

```text
平面
```

高维：

```text
超平面
```

---

## 5. 感知机训练目标

目标：

```text
找到合适的 w 和 b
```

使得所有样本都被正确分类。

例如：

```text
类别1：○
类别2：×

○ ○ ○ ○

----------------

× × × ×
```

感知机需要找到一条线把它们分开。

---

## 6. 感知机学习规则

对于样本：

```math
(x_i,y_i)
```

其中：

```math
y_i \in \{-1,+1\}
```

如果分类错误：

```math
y_i(w^Tx_i+b)\le0
```

则更新参数：

### 更新权重

```math
w=w+\eta y_i x_i
```

### 更新偏置

```math
b=b+\eta y_i
```

其中：

```math
η
```

为学习率（Learning Rate）。

---

## 7. 训练过程示例

假设：

```math
x=[2,1]
```

真实标签：

```math
y=1
```

初始：

```math
w=[0,0]
```

```math
b=0
```

### 前向计算

```math
z=w^Tx+b
```

```math
=0
```

输出：

```math
\hat y=0
```

分类错误。

---

### 参数更新

学习率：

```math
η=0.1
```

更新：

```math
w=w+ηyx
```

```math
=[0,0]+0.1×1×[2,1]
```

```math
=[0.2,0.1]
```

偏置：

```math
b=0+0.1=0.1
```

下一轮继续训练。

---

## 8. 感知机损失函数

感知机损失：

```math
L(w,b)
=
-\sum_{i\in M}
y_i(w^Tx_i+b)
```

其中：

```math
M
```

表示：

```text
所有被错误分类的样本集合
```

目标：

```math
L(w,b)\rightarrow0
```

---

## 9. 感知机为什么能收敛

感知机收敛定理：

如果数据：

```text
线性可分
```

则感知机一定能够在有限步内找到一个分类超平面。

即：

```text
训练一定会结束
```

---

## 10. 感知机的缺点

### 缺点1：只能解决线性可分问题

例如：

```text
○ ○ ○

× × ×

○ ○ ○
```

无法用一条直线分开。

---

### 缺点2：无法解决 XOR 问题

异或（XOR）：

```text
0 XOR 0 = 0
0 XOR 1 = 1
1 XOR 0 = 1
1 XOR 1 = 0
```

分布：

```text
(0,1)     (1,1)
   ●         ○

   ○         ●
(0,0)     (1,0)
```

无法用一条直线划分。

因此：

```text
单层感知机无法解决 XOR
```

---

### 缺点3：表达能力有限

只能学习：

```text
线性关系
```

无法学习：

```text
复杂非线性关系
```

---

## 11. 多层感知机（MLP）

为了克服感知机的缺陷，引入隐藏层：

```text
输入层
   ↓
隐藏层
   ↓
输出层
```

结构：

```text
x
↓
Linear
↓
ReLU
↓
Linear
↓
Output
```

这就是：

```text
多层感知机（MLP）
```

也是现代神经网络的基础。

---

## 12. 感知机与逻辑回归区别

| 对比项 | 感知机 | 逻辑回归 |
|----------|----------|----------|
| 输出 | 0/1 | 概率 |
| 激活函数 | Step | Sigmoid |
| 损失函数 | 感知机损失 | 交叉熵 |
| 是否可导 | 不可导 | 可导 |
| 训练方式 | 误分类更新 | 梯度下降 |
| 应用 | 早期分类器 | 现代分类器 |

---

## 13. 感知机与神经网络关系

发展路线：

```text
感知机
   ↓
多层感知机(MLP)
   ↓
神经网络
   ↓
深度神经网络(DNN)
   ↓
深度学习
```

因此：

> 感知机是现代深度学习的起点。

---

## 14. 总结

### 核心公式

线性计算：

```math
z=w^Tx+b
```

激活函数：

```math
y=f(z)
```

参数更新：

```math
w=w+\eta yx
```

```math
b=b+\eta y
```

---

### 核心特点

✅ 最早的神经网络模型

✅ 能解决线性可分问题

✅ 训练简单

✅ 是神经网络的基础

❌ 无法解决 XOR 问题

❌ 无法学习复杂非线性关系

❌ 表达能力有限

---

一句话总结：

> 感知机就是一个通过学习权重和偏置来寻找分类超平面的线性二分类模型，它是现代神经网络和深度学习的起点。
