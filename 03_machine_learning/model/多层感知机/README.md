# 多层感知机（MLP）介绍

## 1. 什么是多层感知机

多层感知机（Multi-Layer Perceptron，MLP）是一种最基础的前馈神经网络。

它是在单层感知机的基础上增加一个或多个隐藏层，从而能够学习复杂的非线性关系。

结构如下：

```text
输入层
   ↓
隐藏层
   ↓
输出层
```

例如：

```text
x1 ─┐
x2 ─┼─► Hidden Layer ─► Output
x3 ─┘
```

---

## 2. 为什么需要多层感知机

单层感知机只能解决：

```text
线性可分问题
```

例如：

```text
○ ○ ○ ○

-----------

× × × ×
```

可以用一条直线分开。

---

但是对于 XOR 问题：

```text
(0,1) ●      ○ (1,1)

(0,0) ○      ● (1,0)
```

无法使用一条直线分开。

因此：

```text
单层感知机失效
```

---

解决方案：

```text
增加隐藏层
```

即：

```text
感知机
    ↓
多层感知机
```

---

## 3. MLP结构

一个简单的 MLP：

```text
输入层(2)
     ↓
隐藏层(3)
     ↓
输出层(1)
```

示意图：

```text
x1 ──┐
      ├── h1 ──┐
x2 ──┤         │
      ├── h2 ──┼── y
      │         │
      └── h3 ──┘
```

---

## 4. 数学模型

### 第一层

输入：

```math
X=[x_1,x_2,...,x_n]
```

计算：

```math
H=XW_1+b_1
```

其中：

```math
W_1
```

为第一层权重矩阵。

---

### 激活函数

加入非线性：

```math
A=f(H)
```

例如：

```math
ReLU
```

```math
Sigmoid
```

```math
Tanh
```

---

### 第二层

输出层：

```math
Y=AW_2+b_2
```

整体：

```math
Y=f(XW_1+b_1)W_2+b_2
```

---

## 5. 为什么需要激活函数

如果没有激活函数：

```math
Y=(XW_1+b_1)W_2+b_2
```

整理：

```math
Y=XW+b
```

本质上仍然是：

```text
线性模型
```

---

因此：

```text
多层线性 = 单层线性
```

没有意义。

---

激活函数引入非线性：

```text
线性
 ↓
激活函数
 ↓
非线性
```

使网络能够拟合复杂规律。

---

## 6. 常见激活函数

### Sigmoid

公式：



特点：

```text
输出范围：
0 ~ 1
```

优点：

```text
输出可解释为概率
```

缺点：

```text
梯度消失
```

---

### Tanh

公式：

```math
tanh(x)
=
\frac{e^x-e^{-x}}
     {e^x+e^{-x}}
```

特点：

```text
输出范围：
-1 ~ 1
```

---

### ReLU

公式：

```math
ReLU(x)=max(0,x)
```

特点：

```text
x > 0 输出 x
x < 0 输出 0
```

优点：

```text
计算简单
训练速度快
```

目前最常用。

## 8. 损失函数

分类问题常用：

### 交叉熵损失

```math
L=-\sum y\log(\hat y)
```

作用：

```text
衡量预测值与真实值差距
```

---

损失越小：

```text
预测越准确
```

---

## 9. 反向传播

训练核心：

```text
Forward
↓
Loss
↓
Backward
↓
Update
```

---

步骤：

### 1. 前向传播

计算：

```math
\hat y
```

---

### 2. 计算损失

```math
Loss
```

---

### 3. 反向传播

计算：

```math
\frac{\partial Loss}
     {\partial W}
```

---

### 4. 更新参数

梯度下降：

```math
W=W-\eta\frac{\partial L}{\partial W}
```

其中：

```math
η
```

为学习率。

## 12. MLP优缺点

### 优点

✅ 结构简单

✅ 理论成熟

✅ 可以拟合非线性关系

✅ 是深度学习基础

---

### 缺点

❌ 参数量大

❌ 容易过拟合

❌ 不适合图像空间特征提取

❌ 不适合序列建模

---

## 13. MLP与CNN区别

| 对比项 | MLP | CNN |
|----------|----------|----------|
| 输入 | 向量 | 图像 |
| 参数量 | 大 | 小 |
| 特征提取 | 差 | 强 |
| 平移不变性 | 无 | 有 |
| 图像分类 | 一般 | 优秀 |

## 14. MLP与深度学习关系

发展路线：

```text
感知机
   ↓
多层感知机
   ↓
深层神经网络
   ↓
CNN
   ↓
RNN
   ↓
Transformer
```

MLP 是现代深度学习网络的基础结构。

几乎所有神经网络内部都包含：

```text
Linear
+
Activation
```

这一 MLP 核心思想。
