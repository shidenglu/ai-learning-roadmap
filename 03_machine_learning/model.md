# 模型

## 为什么会有模型

问：一个苹果从楼上掉下去，多久可以落地

理论上来讲根据自由落体公式可以迅速求解，但是现实的是，这个世界很复杂，很有因素会影响最后结果的计算，如下面所示

```text
重力
空气阻力
风速
温度
湿度
苹果形状
苹果旋转
```

如果将这些全部考虑进去，人根本算不出来

试想：如果考虑到了上面所有的因素，是不是可以构建一个跟上面所有因素相关的函数，知道具体的外界情况的时候，就可以求解出来具体的下落时间了。

## 什么是模型

模型（Model）是一个参数化函数（Parameterized Function），用于近似输入与输出之间的映射关系。

## 模型的种类

### 机理模型

有具体的理论公式一句，例如牛顿第二定律，根据输入来求解输出

```c
F=ma
```

### 数据模型

**特点：不知道规律，让机器自己寻找规律，得到输入与输出对应关系的模型，这就是机器学习**
**科学家放弃了从已经模型预测结果的想法，进而转变思维。假设存在一个模型，根据真实的输入输出，求解模型里面的具体参数**

## 模型内部到底是什么

**模型内部=公式+参数**

可以理解为，一元二次函数，一元三次，三角函数等，其实都是一种模型，这种是比较简单的模型，比较复杂的模型可能人就无法直观的感受了，因为数量级太庞大了

**合适的问题找到合适的模型很重要**

## 训练

**训练就是确定当前模型内部参数具体的值，其实就是参数估计的过程**

## 如何训练

### 损失函数

```c
L=(y−y^​)2
```

构建误差和参数的函数，找到使误差下降最快的值，更新原来的参数【求偏导数，即当前误差相对特性参数下降最快的方向】

**学习率：梯度更新的速度，太大的话容易导致找不收收敛点，震荡；太小的话，训练的效率可能会低下**

## 模型的组成

